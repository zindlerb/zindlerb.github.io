SVG Transformations

Translate

Translate is the simplest of the SVG transformations. It takes two arguments x and y. When you translate an svg element you simply move the x and y origin by the amount specified with the x and y parameters. Now the shape is drawn relative to the new x and y origin. Try dragging around the gray dot to get a feel for how translate works. 

var svgContainer = d3.select("body").append("svg")
                            .attr("width",200)
                            .attr("height",200);
svgContainer.append("circle")
            .attr("transform", translate(, ))
            .attr("cx", function (d) { return ; })
            .attr("cy", function (d) { return ; })
            .attr("r", function (d) { return 5; })
            .style("fill", function (d) { return "purple"; })

Scale

The scale svg transform takes two arguments x and y. These arguments determine how much each of the axis will be scaled by. Try setting the x scale down low, and the y scale up high. Drag the grey knobs to scale the x and y axis. Observe the difference in speeds when you drag horizontally vs. vertically. 

var svgContainer = d3.select("body").append("svg")
                            .attr("width",200)
                            .attr("height",200);
svgContainer.append("circle")
            .attr("transform", scale(, ))
            .attr("cx", function (d) { return ; })
            .attr("cy", function (d) { return ; })
            .attr("r", function (d) { return 5; })
            .style("fill", function (d) { return "purple"; })

Rotate

Rotate takes 3 arguments: a rotation angle, and optional x and y points to rotate around. If the x and y points are not specified the shape will rotate around the origin. Grab the gray knob to rotate the example shape. 

var svgContainer = d3.select("body").append("svg")
                            .attr("width",200)
                            .attr("height",200);
svgContainer.append("circle")
            .attr("transform", rotate(, , ))
            .attr("cx", function (d) { return ; })
            .attr("cy", function (d) { return ; })
            .attr("r", function (d) { return 5; })
            .style("fill", function (d) { return "purple"; })

skewX

SkewX is a slightly confusing transformation. It only takes one argument. We can think of the argument it takes as the number of degrees we will tilt the y axis. Our shape is then drawn with this tilted y axis. Having our y axis tilted causes the skewing effect we see. To get a better feel for how skew works try tilting the y axis and dragging the shape directly up and down. 

var svgContainer = d3.select("body").append("svg")
                            .attr("width",200)
                            .attr("height",200);
svgContainer.append("circle")
            .attr("transform", skewX())
            .attr("cx", function (d) { return ; })
            .attr("cy", function (d) { return ; })
            .attr("r", function (d) { return 5; })
            .style("fill", function (d) { return "purple"; })

skewY

SkewY is exactly like skewX except we are skewing along the Y axis instead of the X axis. 

var svgContainer = d3.select("body").append("svg")
                            .attr("width",200)
